## Computer-Graphics-Basic
- **언어:** C / C++
- **라이브러리:** OpenGL / GLUT
- **목표:** 3D 좌표계, 행렬 연산, 카메라 인터페이스 등 로우레벨 그래픽스 기초 습득

## 📜 Implementation Details
1. **Kinematics (계층적 구조):** - `glPushMatrix`, `glPopMatrix`를 활용한 로봇 팔의 계층적 움직임 구현. 
2. **Viewports & Navigation:** - 4분할 화면 구성 및 마우스 트랙볼(Trackball) 인터페이스 구현으로 3D 공간 제어 로직.
3. **Geometric Modeling:** - 도넛(Torus) 형태 구현.
